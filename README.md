# board-game-companion
Board Game Companion (also known as Game Game Collector) is simple app to track games on Board Game Geek.

## Functionalities
- Add / Remove game
- Track it ranking
- Save it location/s

## Project Info
- **Android API Level** 24 - 7.0 Nougat
- **Build System** Gradle
- **Deps** No
